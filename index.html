A basic HTML lay out should include the following

  1. HTML Tags
  2. Head
  3. Body

The purpose of the HTML Tags is to let the text editor know what type of document you are drafting

The Head should contain additional information such as where to find the styling information, specific instructions for screen width, etc.

The Body portion is where you will find text and images
